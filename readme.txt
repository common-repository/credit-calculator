== Description ==

Plugin to calculate the cost of credit and percent.

Plug-in "Loan Calculator" designed to calculate the value of annuitentnyh and differentiated credit.

Plugin features:

Input fields for the user: the currency of the loan, the loan amount, loan term, interest rate, down payment, a lump sum fee, monthly kommisiya, the method of repayment.
The possibility of calculating the loan in various currencies
Setting up the initial values ??for the tabs (set in the code of the plugin)
Configuring additional payments for each tab
Ability to print the calculation results
Export calculation results in PDF file.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place shortcode [credit_calculator] in your page.
